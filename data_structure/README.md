### 1. Array vs LinkedList
  - Array
    - 가장 기본적이 자료구조
    - 논리적 저장 순서와 물리적 저장 순서가 일치한다.
    - 인덱스를 통해 요소에 접근할 수 있다. 
  - LinckedList
    - 각각의 원소들은 자기 자신 다음에 어떤 원소인지만을 기
    - Array와 달리 논리적 저장 순서와 물리적 저장 순서가 다르다.
